- Цепочка: последовательность последующих действий в формате: [номер].[действие]([успех],[неудача])

Формат цепочки:
- Каждое звено записывается через точку с запятой ;
- В скобках указываются номера звеньев для перехода:
  - Первое число в скобках — номер звена при успешном выполнении текущего действия
  - Второе число в скобках — номер звена при неудачном выполнении текущего действия
  - 0 означает конец цепочки (выход)
- Строго ровно 2 звена в цепочке. Цепочки из 3, 4 и более звеньев недопустимы.

Примеры:
1.Сообщает о проблеме(2,0);2.Запрашивает подтверждение(0,0) — при успехе Сообщает о проблеме переходит на звено 2 (Запрашивает подтверждение), при неудаче — выход; Запрашивает подтверждение в любом случае завершает цепочку
В скобках только 0 или 2. Запрещено писать (0,3), (3,0) и т.п. — программа отклонит строку. Пример: 1.Запрашивает подтверждение(2,0);2.Сообщает о завершении(0,0)

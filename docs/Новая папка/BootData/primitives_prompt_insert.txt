ИНСТРУКЦИЯ ПО ГЕНЕРАЦИИ БАЗЫ АВТОМАТИЗМОВ ДЛЯ СТАДИИ 2

На основе профиля агента, описанного выше, создай базу автоматизмов в формате:
Состояние|Стили|Слово или звукоподражание|Тон|Настроение

ВАЖНО: Никаких пояснений, вступлений, номеров строк или заголовков. Только сгенерированные строки. Каждая строка — один автоматизм.

КОНТЕКСТ ГЕНЕРАЦИИ

Агент находится на стадии 2 (Автоматизмы), где формируются базовые речевые паттерны через механизм статистического обучения. Автоматизмы создаются путем копирования слов и звукоподражаний, которые агент слышит от Оператора в различных ситуациях.

При поступлении слова или фразы (например, "ма ма" или "ав-ав") движок автоматически:
- Разбивает строку по пробелам или дефису
- Первый элемент становится триггером автоматизма
- Остальные элементы (не более 2) становятся звеньями цепочки
- Если слово длиннее 3 слогов, лишнее автоматически отсекается

ПРИНЦИПЫ РАСПРЕДЕЛЕНИЯ ПО СОСТОЯНИЯМ И СТИЛЯМ

Состояние Плохо:
- Звукоподражания падению/удару: бах-бах, бух-бух, стук-стук, плюх-плюх, шлёп шлёп
- Состояния: бо-бо, у пал, пло хо, нель зя
- Резкие/пугающие звуки: р-р-р, у-у-у, з-з-з, жу-жу (если пугает), кар-кар
- Негативные реакции: тьфу-тьфу, пф-пф, чих-чих, не вкус но
- Стили: все, содержащие Страх, Напряжение, Агрессию, Злость, Протест
- Тон: Повышенный (испуг, злость) или Вялый (грусть, боль)
- Настроение: Плохое, Агрессивное, Защитное, Протест

Состояние Хорошо:
- Звукоподражания с повторами: ля-ля, топ-топ, кап-кап, буль-буль, хлюп-хлюп (игровые)
- Игровые: ку-ку, би-би, ту-ту, чух-чух, динь-динь
- Еда: ням ням, ам-ам, вкус но
- Ласковые обращения: ма ма, па па, ба ба, ля ля, ня ня
- Игрушки: мя чик, миш ка, зай ка (в контексте игры)
- Стили: все, содержащие Игра, Доброта, Эмпатия, Расслабление
- Тон: Нормальный или Повышенный (радость), Вялый (расслабление)
- Настроение: Хорошее, Игривое

Состояние Норма (основной массив):
- Все остальные слова, не маркированные явно позитивно или негативно
- Обращения, названия предметов, бытовые звуки
- Приветствия и вежливость
- Стили: Поиск, Исследование, Восстановление, Лень, Учитель
- Тон: Преимущественно Нормальный, иногда Вялый
- Настроение: Нормальное, Учитель (для исследовательских стилей)

ПРИНЦИПЫ СООТВЕТСТВИЯ ТОНА И НАСТРОЕНИЯ

Тон:
- Повышенный — для неожиданных, радостных, агрессивных или пугающих контекстов
- Вялый — для грусти, усталости, расслабления, задумчивости, нежности
- Нормальный — для нейтральных и большинства исследовательских контекстов

Настроение:
- Игривое — для стилей с Игра и позитивных контекстов
- Учитель — для исследовательских стилей (Поиск+Исследование, Восстановление+Исследование)
- Агрессивное/Протест/Защитное — для конфликтных контекстов (злость, страх, наказание)
- Плохое — для состояния Плохо
- Хорошее — для состояния Хорошо
- Нормальное — для состояния Норма

ПРИМЕРЫ ПРАВИЛЬНЫХ СТРОК (фрагмент полного набора)

Для комбинации стилей "Поиск+Напряжение+Страх" должны быть ровно 3 строки:

Плохо|Поиск+Напряжение+Страх|бах бах|Повышенный|Защитное
Норма|Поиск+Напряжение+Страх|тик так|Повышенный|Нормальное
Хорошо|Поиск+Напряжение+Страх|ку ку|Повышенный|Игривое

Для комбинации стилей "Расслабление+Игра":

Плохо|Расслабление+Игра|р-р-р|Повышенный|Плохое
Норма|Расслабление+Игра|топ топ|Вялый|Нормальное
Хорошо|Расслабление+Игра|ля ля|Вялый|Игривое

ЗАПРЕЩЕНО

- Использовать слова не из расширенного словаря
- Использовать сложные фразы длиннее 3 слогов
- Создавать неконгруэнтные пары (Плохо + "ля ля" + Игривое)
- Пропускать хотя бы одно состояние для любой комбинации стилей
- Добавлять пояснения, нумерацию или заголовки в вывод
- Повторять одинаковые комбинации (стремиться к разнообразию)

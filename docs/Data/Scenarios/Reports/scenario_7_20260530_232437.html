НазваниеСуммация (оба рефлекса при близкой силе) условных рефлексов.
ОписаниеТестируется формирование первичных условных рефлексов, суммация (оба рефлекса при близкой силе) на 1 стадии развития агента.

Образ восприятия агента включает вербальные (фразы с пульта), тактильные (воздействия с пульта) и зрительные (цвет фона сцены) компоненты; составной (компаундный) стимул может объединять любые из этих модальностей.

ВАЖНО: стимулы компаунда должны принадлежать РАЗНЫМ модальностям (вербальный + зрительный). Два стимула одной модальности (например две фразы) не распознаются системой как компаунд, потому что CompoundModalityCount считает число задействованных каналов (фразы, воздействия, цвет), а не число стимулов.

Механизм:
Если два стимула разных модальностей предъявлены вместе, и каждый имеет условный рефлекс к разным безусловным ответам с близкой крепостью, активируются оба ответа одновременно (смешанный ответ).

Формула:

S₁ + S₂ → R₁ и R₂ (оба ответа активируются)

θ = min(C₁, C₂) / max(C₁, C₂)

Оба ответа активируются при θ ≥ θ_comp (по умолчанию 0.7), то есть когда крепости обоих рефлексов достаточно близки. Каждый рефлекс при этом должен превышать порог активации: C_i ≥ γ.

Экспериментальная база:
— Weiss S.J. (1972). Stimulus compounding in free-operant and classical conditioning — компаунд вызывает смешанные или суммарные реакции. PB, 78(3), 189–208.
— Rescorla R.A. (1997). Summation and overexpectation with qualitatively different outcomes. ALB, 25(1), 50–62.

Протокол тестирования:

Шаг 1. Сформировать CR1: «фраза Б1» (вербальный канал) → безусловный [В1] (например, [Развеселить]). Повторить 4–5 раз. Довести крепость до C₁ ≈ 0.8. Цвет фона при обучении — Белый.
Шаг 2. Сформировать CR2: «цвет Красный» (зрительный канал, без фразы) → безусловный [В2] (например, [Порадовать]). Повторить 4–5 раз. Довести крепость до C₂ ≈ 0.8.
Шаг 3. Предъявить «Б1» + Красный фон одновременно (вербальный + зрительный каналы). Отношение крепостей: θ = 0.8/0.8 = 1.0 ≥ θ_comp = 0.7. Ожидается смешанный ответ: обе реакции ([Смеется] и [Радуется]).

Критерий успеха: оба безусловных ответа активируются одновременно при близких крепостях условных рефлексов (θ ≥ θ_comp).

ВАЖНО: действия б/у рефлексов, используемых в протоколе не должны быть антагонистами. В противном случае активируется одно из них.

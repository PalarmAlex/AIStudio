═══════════════════════════════════════════════════════════════
I. Теоретические основания
═══════════════════════════════════════════════════════════════

Клонирование рефлекса в автоматизм:
Переход от генетически заложенного рефлекса к приобретённой моторной программе. На 2 стадии развития гомеостатический стимул, вместо прямой активации безусловного рефлекса, инициирует создание автоматизма — адаптивной копии рефлекса с теми же пусковыми условиями и действиями. Если исходный рефлекс имеет цепочку, она так же клонируется в цепочку автоматизма. Аналог перехода от врождённых рефлексов к приобретённым навыкам в онтогенезе.

Эхо-автоматизм на вербальный стимул:
Воспроизведение предъявленного вербального стимула в качестве ответного действия. На 2 стадии вербальный стимул анализируется по структуре:

- Однословный стимул → эхо-автоматизм без цепочки: S → S.
- Многословный стимул (фраза) → эхо-автоматизм с цепочкой: первое слово становится действием автоматизма, последующие слова — звеньями цепочки.

Ограничение стадии 2: длина цепочки ограничена двумя звеньями, избыточные элементы фразы отбрасываются.

═══════════════════════════════════════════════════════════════
II. Механизм клонирования рефлекса
═══════════════════════════════════════════════════════════════

При поступлении гомеостатического стимула:

1. Определяется безусловный рефлекс, соответствующий текущим условиям {Состояние, Стили, Триггер}.
2. Создаётся автоматизм с идентичными пусковыми условиями и действием.
3. Если рефлекс имеет цепочку из N звеньев, создаётся цепочка автоматизма (с учётом ограничения стадии на максимальную длину).
4. Автоматизм регистрируется в дереве активации.

При повторном предъявлении того же стимула активируется уже созданный автоматизм, а не исходный безусловный рефлекс.

═══════════════════════════════════════════════════════════════
III. Механизм эхо-автоматизма с разбором фразы
═══════════════════════════════════════════════════════════════

При поступлении вербального стимула фраза сегментируется по пробелам на токены T₁, T₂, ..., T_m:

- Действие автоматизма: T₁ (первый токен).
- Звено 1 цепочки: T₂ (второй токен).
- Звено 2 цепочки: T_m (последний токен).
- Токены T₃..T_m−1 отбрасываются (ограничение 2 стадии).

Пример:
Стимул «ка на рей ка» → токены [ка, на, рей, ка].
Автоматизм: действие «ка» + цепочка: звено 1 «на», звено 2 «ка».
Токен «рей» отброшен. Результирующее воспроизведение: «ка-на-ка».

Для однословного стимула цепочка не создаётся:
Стимул «ага» → автоматизм: действие «ага», без цепочки.

═══════════════════════════════════════════════════════════════
IV. Протокол тестирования
═══════════════════════════════════════════════════════════════

Блок A — клонирование безусловного рефлекса с цепочкой:

Шаг 1. Гомеостатический стимул [Обрадовать]. Состояние → Хорошо, стили изменяются. Вместо прямой активации безусловного рефлекса создаётся автоматизм-клон с цепочкой.
Шаг 2. Пауза. Выполняется звено 1 цепочки автоматизма.
Шаг 3. Пауза. Выполняется звено 2 цепочки автоматизма.

Блок B — эхо-автоматизм с цепочкой (многословная фраза):

Шаг 4. Вербальный стимул «ку ку». Фраза сегментируется: [ку, ку]. Создаётся эхо-автоматизм: действие «ку» + цепочка: звено 1 «ку». Агент отвечает.
Шаг 5. Пауза. Выполняется звено цепочки.

Блок C — эхо-автоматизм без цепочки (одно слово):

Шаг 6. Вербальный стимул «ага». Одно слово — цепочка не создаётся. Создаётся эхо-автоматизм: действие «ага». Агент отвечает.

Критерии успеха:
- Блок A: гомеостатический стимул создаёт автоматизм (не активирует безусловный рефлекс напрямую), цепочка клонируется и выполняется.
- Блок B: многословная фраза порождает эхо-автоматизм с цепочкой, токены корректно распределены между действием и звеньями.
- Блок C: однословная фраза порождает эхо-автоматизм без цепочки.

Руководство автора адаптера среды

Практическая инструкция для разработчика host-среды (Velum, Excel, CAD…). AIStudio не обязательна: ядро — isida.dll, runtime contract YAML (AdapterContract.md v3.2), dispatch рецептов по G_AD на OnPulseCompleted.

Архитектура: SymbiontArchitecture_IsidaAdapter.md. Норматив форматов: AdapterContract.md.

Версия гайда: 2.0 (2026-06-21), contractVersion: 3.2.

Удалено в v3.2: EnvironmentTriggers.yaml, recommended_trigger_keys, expression_pattern_id, IHostMotorDispatcher, trigger-detect.json, trigger-catalog.json.

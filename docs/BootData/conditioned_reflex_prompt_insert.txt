ИНСТРУКЦИЯ ПО ГЕНЕРАЦИИ БАЗЫ УСЛОВНЫХ РЕФЛЕКСОВ

На основе профиля агента, описанного выше, создай базу условных рефлексов в формате:
Состояние|Стили|Триггер безусловного рефлекса|Новый триггер условного рефлекса|Тон|Настроение

ВАЖНО: Никаких пояснений, вступлений, номеров строк или заголовков. Только сгенерированные строки. Каждая строка — один рефлекс.

КОНТЕКСТ ГЕНЕРАЦИИ

Агент взаимодействует с оператором через пульт, который передаёт:

- Триггеры безусловных рефлексов, внешний сигнал с пульта управления из списка: [InfluenceActionList].
- Вербальные возгласы — короткие междометия с пунктуацией
- Тон — физические параметры голоса: Вялый, Нормальный, Повышенный
- Настроение — эмоциональный контекст: Нормальное, Хорошее, Плохое, Игривое, Учитель, Агрессивное, Защитное, Протест

Условные рефлексы формируются в контексте [Состояние + Стили] и связывают безусловное событие с полным пакетом: возглас + тон + настроение, который оператор может использовать в будущем для предсказуемого воздействия на агента.

ОБЪЁМ ГЕНЕРАЦИИ

1. Этап 1. Сформируй полный набор условий запуска рефлекса:
   - Используй ВСЕ [ReflexGenStyleCount] уникальных комбинаций стилей (каждую ровно один раз).
   - Для КАЖДОЙ комбинации стилей создай варианты для ВСЕХ состояний (Плохо, Норма, Хорошо).
   - Для КАЖДОГО сочетания «состояние + стиль» создай варианты для ВСЕХ триггеров из [InfluenceActionList].
   - Общее количество комбинаций на этом этапе: [ReflexGenLinesThreeStates].

2. Этап 2. Из полного набора отбери итоговые рефлексы со следующим распределением:
   - Плохо: ~35 % от общего количества комбинаций для состояния «Плохо» → [ReflexGenLinesPerState] строк;
   - Норма: ~40 % от общего количества комбинаций для состояния «Норма» → [ReflexGenLinesPerState] строк;
   - Хорошо: ~25 % от общего количества комбинаций для состояния «Хорошо» → [ReflexGenLinesPerState] строк.

ОПИСАНИЕ ПОЛЕЙ СТРОКИ

- Состояние — внутреннее состояние агента в момент формирования связи. Допустимые значения: Плохо, Норма, Хорошо
- Стили — текущая активность и контекст обучения. Используй ровно [ReflexGenStyleCount] уникальных комбинаций из списка [stileCombination].
   В полном наборе условий КАЖДАЯ комбинация стилей должна встретиться один раз для каждого состояния (Плохо, Норма, Хорошо).
- Триггер безусловного рефлекса — событие, вызывающее «прошитую» реакцию. Допустимые значения: [InfluenceActionList]
- Новый триггер условного рефлекса — вербальный возглас. Строго из списка, приведённого ниже
- Тон — физическая характеристика голоса. Допустимые значения: Вялый, Нормальный, Повышенный
- Настроение — эмоциональный контекст воздействия. Допустимые значения: Нормальное, Хорошее, Плохое, Игривое, Учитель, Агрессивное, Защитное, Протест

ПРИНЦИПЫ СООТВЕТСТВИЯ

Конгруэнтность по валентности (обязательно):

Негативные события — триггеры: Покалечить, Напугать, Огорчить, Опечалить, Истощить, Наказать
- Возгласы: Ай!, Ой!, Ох!, Фу!, Тьфу!, Брр!, У-у-у..., Эх...
- Тон: Повышенный (испуг, злость) или Вялый (грусть, истощение)
- Настроения: Плохое, Агрессивное, Протест, Защитное

Позитивные события — триггеры: Поощрить, Обрадовать, Развеселить, Накормить, Вылечить, Успокоить
- Возгласы: Ага!, Ого!, Ура!, Ха-ха!, Ммм!, М-м-м..., Ах...
- Тон: Нормальный или Повышенный (радость), Вялый (расслабление)
- Настроения: Хорошее, Игривое, Нормальное

Эпистемические (познание) — триггер: Удивляется (как реакция)
- Возгласы: Ого!, А?, Ух ты!, Хм?, Что?, Где?
- Тон: Нормальный, Повышенный
- Настроения: Нормальное, Учитель, Игривое

Учет Тона:
- Повышенный — для резких, неожиданных, радостных или агрессивных событий
- Вялый — для грусти, усталости, расслабления, задумчивости
- Нормальный — для нейтральных и большинства исследовательских контекстов

Учет Настроения:
- Настроение должно соответствовать валентности события и контексту стилей
- Игривое — для стилей с Игра и позитивных событий
- Учитель — для исследовательских стилей и реакции Удивляется
- Агрессивное/Протест/Защитное — для конфликтных контекстов (злость, страх, наказание)

ПРИМЕРЫ ПРАВИЛЬНЫХ СТРОК

Плохо|Поиск+Напряжение+Страх|Покалечить|Ай!|Повышенный|Плохое
Норма|Поиск+Исследование|Удивляется|А?|Нормальный|Нормальное
Хорошо|Расслабление+Игра|Развеселить|Ха-ха!|Повышенный|Игривое

ЗАПРЕЩЕНО

- Использовать сложные фразы («молодец», «хороший мальчик», «дай лапу»)
- Использовать возгласы не из списка
- Создавать неконгруэнтные пары (например, Покалечить + Ха-ха + Повышенный + Игривое)
- Использовать возгласы без пунктуации (все строки должны иметь !, ?, ... или их комбинации)
- Добавлять пояснения, нумерацию или заголовки в вывод
- Повторять одинаковые комбинации (стремиться к разнообразию в рамках правил)
